# All-Agent Golden Example

This public example is the single source of truth for PipeBuilder's static all-agent E2E input
and its human-reviewed golden expectations.

```text
all-agents-golden/
├── space/       # PipeSpace declaration and space-level agent sources
├── providers/   # reusable Skill Provider input
├── projects/    # project referenced by the workspace
└── expected/    # independently reviewed generated targets and full-file goldens
```

Tests copy `space/`, `providers/`, and `projects/` into an isolated temporary Sandbox. They
read `expected/` directly from this example, so expected data is never generated by the
production implementation during a test.

From the repository root, run:

```bash
python3 tests/e2e/run.py --tier offline --case GoldenBuildCases
```
